## 📊 Visualizations Included

The repository contains code to generate several distinct chart types, 
each serving a unique analytical purpose:

| Chart Type | Analytical Purpose |
|------------|-------------------|
| **Line Plot** | Tracks daily fluctuations and identifies specific trends throughout the week |
| **Stacked Bar Plot** | Displays the daily cumulative contribution of each rating category |
| **Scatter Plot** | Explores the correlation between specific attributes, such as Flavor, and the Overall rating |
| **Histogram** | Visualizes the frequency distribution of scores across the entire dataset |
| **Box Plot** | Shows data distribution while highlighting medians and consistency (variability) |
| **Area Chart** | Illustrates the magnitude of change for all categories stacked over time |
| **Pie Chart** | Provides a breakdown of how the Overall rating is distributed by day |
